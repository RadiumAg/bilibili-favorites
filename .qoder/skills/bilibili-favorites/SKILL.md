# Bilibili Favorites Extension Guidelines

Comprehensive development guide for the Bilibili Favorites Chrome extension. Contains 5 rule categories covering project standards, API integration, Chrome extension patterns, React components, and data analysis features.

## When to Apply

Reference these guidelines when:

- Working on any code in this Chrome extension project
- Integrating Bilibili or OpenAI APIs
- Building or modifying React components (popup, options, content script)
- Implementing Chrome extension features (messaging, storage, manifest)
- Developing data analysis and ECharts visualization features
