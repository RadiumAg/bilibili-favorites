# 隐私声明

**最后更新日期：2026年2月10日**

## 概述

Bilibili 收藏夹管理助手（以下简称"本扩展"）致力于为用户提供便捷的 Bilibili 收藏夹管理服务。我们高度重视您的隐私，本隐私声明将向您说明本扩展如何处理您的数据。

## 数据收集

### 本扩展**不收集**以下任何用户信息：

- ❌ 不收集个人身份信息（姓名、邮箱、电话等）
- ❌ 不收集浏览器指纹或设备标识
- ❌ 不收集用户浏览历史记录
- ❌ 不收集使用行为统计数据
- ❌ 不收集任何形式的遥测数据或分析数据

## 本地数据处理

本扩展仅在您的浏览器本地进行数据处理，具体包括：

### 1. Bilibili API 数据访问
- 本扩展仅访问 Bilibili 官方提供的公开 API 接口，用于获取您已登录账号的收藏夹信息
- 所有数据请求均在您已登录 Bilibili 的浏览器环境中发起
- 访问范围仅限于您本人的收藏夹数据，不会访问其他用户的任何数据

### 2. 本地存储
本扩展使用以下本地存储方式：
- **IndexedDB**：用于缓存收藏夹分析数据，缓存有效期为24小时，以减少网络请求
- **Chrome Storage**：用于存储用户的配置偏好设置（API Key、模型选择、关键词规则等）

### 3. 浏览器权限
本扩展请求以下 Chrome 扩展权限：

| 权限 | 用途 | 数据处理 |
|------|------|---------|
| `storage` | 存储用户配置和缓存数据 | 数据仅保存在浏览器本地，不上传至任何服务器 |
| `tabs` | 读取 Bilibili 页面标签信息 | 仅用于获取 B 站页面的基本信息，不读取其他网站 |
| `activeTab` | 访问当前激活标签页 | 仅用于在 Bilibili 页面上注入内容脚本，不访问其他页面 |

**重要说明**：所有存储数据仅保存在您的浏览器本地，不会上传到任何第三方服务器。

### 3. OpenAI API（可选功能）
- 如果您使用基于 GPT 的视频标题关键词提取功能，本扩展会向 OpenAI API 发送视频标题文本
- 数据传输直接从您的浏览器到 OpenAI 官方服务器，本扩展不经过任何中转服务器
- 您可以随时选择不使用该功能

## 第三方服务

本扩展可能使用以下第三方服务：

| 服务名称 | 用途 | 数据处理 |
|---------|------|---------|
| Bilibili API | 获取收藏夹数据 | 数据直接从您的浏览器发送至 Bilibili 官方服务器 |
| OpenAI API（可选） | 视频标题关键词提取 | 数据直接从您的浏览器发送至 OpenAI 官方服务器 |

**第三方服务的数据处理均受其各自隐私政策的约束，我们不对第三方服务的隐私实践负责。**

## 数据共享

我们**不会**在以下任何情况下共享您的数据：

- ❌ 不会向广告商或营销机构共享数据
- ❌ 不会向数据代理商出售或租赁数据
- ❌ 不会与其他第三方服务共享数据（除上述必须使用的 API 服务外）
- ❌ 不会在合并、收购或资产转让时转让用户数据

## 数据安全

为确保您的数据安全，我们采取以下措施：

- 所有数据仅存储在您本地的浏览器中
- 不设立任何后端服务器或数据库
- 使用 HTTPS 协议进行所有网络通信
- 遵循 Chrome Extension Manifest V3 的安全最佳实践

## 用户权利

您拥有以下权利：

- **查看权**：您可以随时查看浏览器中存储的所有数据
- **删除权**：您可以通过清除浏览器数据或卸载本扩展来删除所有本地存储的数据
- **退出权**：您可以随时停止使用本扩展，通过浏览器扩展管理页面卸载

## 儿童隐私

本扩展不针对13岁以下儿童设计，也不会故意收集13岁以下儿童的个人信息。如果发现我们无意中收集了此类信息，我们将采取措施及时删除。

## 隐私声明变更

我们可能会不时更新本隐私声明。重大变更时，我们会通过扩展更新或适当方式通知您。建议您定期查阅本声明以了解最新信息。
