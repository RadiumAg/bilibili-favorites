## 📊 收藏内容分析

深度分析你的 B 站收藏内容，让你对自己的兴趣分布一目了然：

- **智能分析**：深度分析收藏内容分布情况，自动识别内容类别
- **AI 驱动**：基于 GPT 的视频标题关键词提取，实现自动分类
- **可视化展示**：直观展示收藏内容类别占比和趋势图表
- **最近7天统计**：快速了解最近一周的收藏动态
- **数据缓存**：智能缓存机制（24小时有效期），减少重复请求，提升性能

## 🗂️ 智能整理

告别手动一个个移动视频的烦恼，批量高效整理你的收藏夹：

- **一键整理**：快速整理默认收藏夹中的视频内容
- **批量操作**：支持批量移动视频到指定收藏夹
- **自定义规则**：根据关键词匹配自动归类
- **关键词管理**：灵活配置和管理分类关键词
- **🤖 AI 智能整理**：使用 AI 自动分析视频标题，智能分类到对应收藏夹

## 🖱️ 可视化拖拽管理

像管理文件夹一样直观地管理你的收藏夹：

- **直观操作**：左侧显示收藏夹列表，右侧显示视频列表
- **多选支持**：支持 `Ctrl / Cmd` + 点击多选视频
- **拖拽移动**：选中视频后直接拖拽到目标收藏夹即可完成移动
- **批量处理**：一次性移动多个视频，高效便捷
- **实时反馈**：移动完成后自动刷新列表，显示操作结果

## 📌 侧边栏模式

在浏览 B 站的同时，随时整理收藏夹，两不耽误：

- **持久显示**：在浏览器侧边栏中打开，不会因点击其他地方而消失
- **边浏览边操作**：可以一边浏览 B 站视频，一边整理收藏夹
- **更大空间**：比弹窗模式有更大的操作空间

**两种打开方式：**

1. 右键点击扩展图标 → 选择「在侧边栏中打开」
2. 点击浏览器侧边栏图标 → 选择本扩展

## 📦 安装

### 方式一：Chrome Web Store（推荐）

访问 [Chrome Web Store - B站收藏夹整理工具](https://chromewebstore.google.com/detail/b%E7%AB%99%E6%94%B6%E8%97%8F%E5%A4%B9%E6%95%B4%E7%90%86%E5%B7%A5%E5%85%B7/kompiebhklojoioghbddednmmnebnkoc)，点击「添加到 Chrome」即可完成安装。

### 方式二：本地安装

1. 下载最新版本的扩展压缩包并解压
2. 打开 Chrome，进入扩展管理页面 `chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」，选择解压后的文件夹

## 🚀 快速上手

1. 点击浏览器工具栏中的扩展图标打开弹窗
2. 确保已登录 Bilibili 账号，并保持至少一个 B 站页面处于打开状态
3. 查看收藏夹分析数据，了解你的内容分布
4. 根据需要进行收藏夹整理操作

<Callout type="info">
  如果提示需要登录，请刷新 B 站页面后重新打开插件。
</Callout>

## 🤖 AI 功能配置

本扩展支持接入 AI 大模型，实现更智能的关键词提取和自动分类：

1. 进入扩展设置页面，切换到「配置」Tab
2. 填写你的 OpenAI API Key 并选择模型
3. 在「关键词管理」中点击「AI 提取关键词」，自动生成分类关键词

<Callout type="tip">
  推荐使用**星火大模型**，因为提供免费额度，非常适合日常使用。
</Callout>

如果不想使用 AI，也可以使用内置的本地算法（基于 **TF-IDF** 算法）进行关键词提取。

## 🔒 隐私说明

本扩展完全尊重你的隐私：

- ✅ **不收集**任何个人身份信息
- ✅ **不上传**用户数据到任何第三方服务器
- ✅ **所有数据**仅存储在你的浏览器本地（IndexedDB + Chrome Storage）
- ✅ **网络请求**仅用于访问 Bilibili 官方 API 和 OpenAI API（可选）
