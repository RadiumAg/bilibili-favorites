## 环境要求

在开始前，请确认你的环境满足以下条件：

- **浏览器**：Chrome 114+ 或任意 Chromium 内核浏览器（Edge、Arc、Brave 等）；
- **账号**：已注册并能正常登录的 Bilibili 账号；
- **网络**：可正常访问 `bilibili.com`；若启用 AI 功能，还需可访问所选模型的 API 端点。

## 安装

推荐从 Chrome Web Store 一键安装；无法访问商店时可使用本地加载方式。

<Tabs items={['Chrome Web Store（推荐）', '本地加载']}>
  <Tab value="Chrome Web Store（推荐）">
    <Steps>
      <Step>
        ### 打开商店页面
        访问 [Chrome Web Store - B 站收藏夹整理工具](https://chromewebstore.google.com/detail/b%E7%AB%99%E6%94%B6%E8%97%8F%E5%A4%B9%E6%95%B4%E7%90%86%E5%B7%A5%E5%85%B7/kompiebhklojoioghbddednmmnebnkoc)。

      <Step>
        ### 添加到 Chrome
        点击 **「添加至 Chrome」** → **「添加扩展程序」** 完成安装。
      </Step>
      <Step>
        ### 固定到工具栏
        点击浏览器右上角的 🧩 拼图图标，将本扩展 **📌 固定** 到工具栏，方便后续快速唤起。

</Tabs>

## 首次配置

安装完成后，建议按以下顺序完成首次配置，以获得完整体验。

<Steps>
  <Step>
    ### 保持 B 站登录态
    在浏览器中打开任意 B 站页面并完成登录，保持至少一个标签页处于打开状态。扩展依赖该页面的登录态调用 Bilibili 官方 API。
  </Step>
  <Step>
    ### （可选）配置 AI 模型
    如需使用 **AI 整理** 与 **AI 关键字提取**，请进入扩展的 **选项页 → 配置** Tab，选择内置服务商（通义千问 / Kimi / OpenAI / GLM）或 **自定义**，填入 **API Key** 与 **模型名称**。

    <Callout type="info">
      若希望零成本体验，可在配置页顶部切换到 **「免费额度」** 模式，使用扩展内置的 AIGate 共享配额，无需申请任何 Key。
    </Callout>
  </Step>
  <Step>
    ### （可选）初始化关键字规则
    进入 **选项页 → 关键字管理** Tab，选择某个收藏夹，在顶部的 **模式选择器** 中选 **「本地算法」**（TF-IDF，完全离线、免费）或 **「AI 智能」**，再点击 **「为选中创建」** / **「为所有创建」** 批量生成候选关键字，后续可人工微调。
  </Step>
</Steps>

## 启动与基本使用

完成安装与配置后，按下面的路径开始使用核心功能。

<Steps>
  <Step>
    ### 唤起扩展
    点击浏览器工具栏中的扩展图标即可打开弹窗；也可以右键图标选择 **「在侧边栏中打开」**，获得更大的操作空间与持久化显示。
  </Step>
  <Step>
    ### 查看收藏分析
    首次打开会自动拉取你的收藏数据并缓存 **24 小时**。在 **收藏夹分析** 页可查看：

    - 各分类占比饼图；
    - 收藏趋势折线图；
    - 最近 7 天新增收藏统计。

    点击 **「刷新」** 可强制重新拉取最新数据。
  </Step>
  <Step>
    ### 一键批量整理
    在 **弹窗（或侧边栏）底部** 有两个整理按钮：

    - **通过关键字整理**：基于本地关键字规则把默认收藏夹的视频分发到各目标收藏夹，不消耗 AI Token；
    - **🤖 AI 整理**：把视频标题发给 AI 智能分类（首次点击弹出 Token 消耗提醒，**5 秒内再次点击** 才会真正执行）。

    执行过程中界面会显示遮罩与 loading，完成后 Toast 汇总 **成功 / 失败 / 兜底** 计数。
  </Step>
  <Step>
    ### 可视化拖拽微调
    进入 **选项页 → 可视化管理** Tab，左侧是收藏夹列表，右侧是当前收藏夹内的视频：

    - 按住 `Ctrl`（macOS 为 `⌘`）+ 点击实现 **多选**；
    - 使用右上角 **「全选」** 按钮快速选中全部；
    - 选中后直接 **拖拽** 到左侧目标收藏夹完成移动；
    - 操作完成后列表自动刷新，并 Toast 展示成功 / 失败数量。
  </Step>
</Steps>

<Callout type="warn">
  若扩展提示 **「需要登录」**，请先刷新任意一个 B 站页面后再重新唤起扩展。更多常见问题与故障排除见 [注意事项](/docs/attention)。
</Callout>
