`v1.2.6` 是扩展当前随 Chrome Web Store 发布的稳定版本。本页基于源码事实列出该版本 **实际已具备** 的能力，避免引入任何尚未实现的描述。

## 核心能力

### 批量整理

弹窗 / 侧边栏底部共提供 4 个按钮：

- **通过关键字整理**：把默认收藏夹的视频按关键字规则分发到目标收藏夹；
- **自动创建关键字**：跳转到「关键字管理」Tab，支持 **本地 TF-IDF** 与 **AI 智能** 两种模式；
- **🤖 AI 整理**：首次点击弹 Token 消耗提醒，5 秒内再次点击才真正调用 AI；
- **跳转到可视化管理**：快速打开选项页的可视化拖拽界面。

### 可视化管理

选项页的独立 Tab，支持：

- **多选**：`Ctrl` / `⌘` + 点击；
- **范围追加**：`Shift` + 点击；
- **全选 / 取消全选**：右上角按钮；
- **拖拽移动**：把选中的视频拖到左侧任意目标收藏夹；拖到源收藏夹自身会被忽略；
- **完成反馈**：Toast 汇总成功 / 失败数量。

### AI 配置

支持两种模式：

- **自定义（custom）**：内置 4 个主流适配器（通义千问 / Kimi / OpenAI / GML）以及「自定义」入口；适配器切换会覆盖 Base URL 与 Extra Params 为默认值；
- **免费额度（free）**：使用 AIGate 共享配额，**无需任何 Key**；配合配置页的「检查配额」查看剩余量。

配置字段：`AI 模型` / `模型` / `API Key` / `Base URL`（可选）/ `Extra Params`（仅 custom 模式可见）；字段修改会 **自动保存**。

### 收藏夹数据分析

选项页的独立 Tab，基于已配置的关键字规则对收藏夹做占比分析。未命中任何关键字的视频归入「未分类」。

## 已知限制（v1.2.6）

<Callout type="warn" title="下列能力在该版本尚未提供">
  - **无执行预览**：`通过关键字整理` 和 `🤖 AI 整理` 都不提供执行前的 Dry Run；
  - **无撤销**：批量移动完成后没有「一键撤销」功能；
  - **无关键字排除词**：关键字是正向包含匹配，暂不支持显式的"排除词"；
  - **无多账号切换**：扩展始终使用当前激活 B 站标签页的登录态；
  - **可视化管理无键盘快捷键**：除了 `Ctrl` / `⌘` / `Shift` + 点击外，不支持上下键 / 空格 / Esc 等操作。
</Callout>

## 权限清单

- `permissions`: `storage`、`tabs`、`sidePanel`；
- `host_permissions`: `api.openai.com`、`*.openai.com`、`spark-api.xf-yun.com`、`*.xf-yun.com`、`*.radiumg.top`。
