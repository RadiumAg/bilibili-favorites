## 版本号约定

扩展遵循 **语义化版本** 风格的 `MAJOR.MINOR.PATCH`：

| 位段 | 含义 | 触发条件 |
|------|------|---------|
| `MAJOR` | 主版本 | 存在不兼容变更（如重新设计存储结构、移除核心功能） |
| `MINOR` | 次版本 | 新增功能、新增 UI、新增适配器等向后兼容的变更 |
| `PATCH` | 修订 | Bug 修复、文案调整、依赖升级等无功能变化 |

## 如何查看当前已安装的版本

<Callout type="info">
  在 `chrome://extensions/` 中找到本扩展，版本号显示在扩展名称右侧（或点击「详情」查看完整信息）。也可以在扩展弹窗中通过「查看使用方式」右上角获取。
</Callout>
