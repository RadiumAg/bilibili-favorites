**性格分析** 是扩展提供的趣味功能，通过分析你的收藏夹内容，为你生成一份专属 **MBTI 性格画像**。

## 你能看到什么

- **MBTI 类型**：AI 根据你的收藏偏好推断出的 MBTI 四字母类型；
- **四维度进度条**：外向/内向、感觉/直觉、思考/情感、判断/感知，每个维度的倾向程度可视化展示；
- **兴趣标签云**：从收藏内容中提取的高频关键词，直观呈现你的兴趣画像；
- **个性化建议**：基于分析结果的趣味性格解读和建议。

## 使用步骤

<Steps>
  <Step>### 进入性格分析页 在选项页中切换到 **「性格分析」** 标签页。</Step>
  <Step>
    ### 等待数据加载
    首次打开会自动拉取你的全部收藏数据，加载过程中会显示当前进度。数据会在分析页与性格分析页之间共享，无需重复加载。
  </Step>
  <Step>### 开始分析 点击 **「开始分析」** 按钮，AI 将开始分析你的收藏偏好。</Step>
  <Step>
    ### 查看结果 分析完成后，你将看到完整的 MBTI
    性格画像，包括类型、维度、标签云和个性化建议。如需重新分析，点击 **「重新分析」** 即可。
  </Step>
</Steps>

## AI 模式

性格分析支持两种 AI 模式，可在 [配置页](/docs/configuration) 中切换：

| 模式            | 说明                                                                |
| --------------- | ------------------------------------------------------------------- |
| **内置免费 AI** | 使用扩展提供的 AIGate 服务，无需配置，开箱即用                      |
| **自定义 AI**   | 填入你自己的 OpenAI 兼容 API Key 和模型，支持 OpenAI、星火、Kimi 等 |

<Callout type="info">
  **内置模式有额度限制吗？**

内置 AIGate 服务提供免费额度，每日可用次数有限。如需更高频使用，建议切换到自定义 AI 模式。

</Callout>

## 数据来源

- **输入数据**：仅使用你收藏夹中的视频标题、分类分布和高频关键词，**不会上传完整视频内容**；
- **分析模型**：AI 模型在服务端完成推理，本地不做模型计算；
- **结果存储**：分析结果保存在本地，不会上传到任何服务器。

## 常见疑问

<Callout type="info">
  **「分析结果准确吗？」**

性格分析是一种趣味性功能，结果基于收藏内容的统计特征和 AI 推断，仅供参考和娱乐，不构成专业心理评估。

</Callout>

<Callout type="warn">
  **「分析失败怎么办？」**

请检查：1) 是否已登录 B 站账号；2) AI 配置是否正确；3) 网络是否通畅。如使用自定义 AI，请确认 API Key 有效且有足够额度。

</Callout>
