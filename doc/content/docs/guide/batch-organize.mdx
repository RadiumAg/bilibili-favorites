**批量整理** 是扩展最核心的自动化能力：扩展会读取你 **默认收藏夹** 中的视频，基于 [关键字规则](/docs/guide/keyword-management) 将每条视频自动移动到命中的目标收藏夹。

## 入口

批量整理的入口在 **弹窗（Popup）** 与 **侧边栏（SidePanel）** 底部，**不在选项页**。共有两种触发方式：

- **通过关键字整理**：基于本地关键字规则分类（不消耗 AI Token）；
- **🤖 AI 整理**：把所有视频标题发送给 AI 做智能分类（消耗 Token，效果通常更好）。

## 方式一：通过关键字整理

<Steps>
  <Step>
    ### 唤起弹窗或侧边栏
    点击浏览器工具栏中的扩展图标打开弹窗；或右键图标 → **「在侧边栏中打开」**。
  </Step>
  <Step>
    ### 确认默认收藏夹已设置
    扩展会把 **默认收藏夹** 作为固定的源收藏夹。若尚未设置，点击任意收藏夹右侧按钮将其标记为默认。
  </Step>
  <Step>
    ### 确认关键字规则
    确保每个目标收藏夹都已经配置了一组关键字。未配置的收藏夹在本次整理中不会被作为归类目标。

    如需调整，点击 **「自动创建关键字」** 按钮跳转到选项页的 **关键字管理** Tab；具体见 [关键字管理](/docs/guide/keyword-management)。
  </Step>
  <Step>
    ### 点击「通过关键字整理」
    弹窗底部的第一个按钮即为 **「通过关键字整理」**。按钮旁的 `?` 可展开说明浮层。
  </Step>
  <Step>
    ### 查看结果
    扩展会在遮罩层中显示执行状态；完成后 Toast 会提示成功 / 失败数量，并刷新本地缓存。
  </Step>
</Steps>

## 方式二：AI 整理

<Steps>
  <Step>
    ### 完成 AI 配置
    先在 [API Key 配置](/docs/configuration/api-key) 中完成 **自定义配置** 或切换到 **免费额度** 模式；未配置时点击按钮会弹出提示并自动打开配置页。
  </Step>
  <Step>
    ### 点击「🤖 AI 整理」
    弹窗底部的第三个按钮即为 **「🤖 AI 整理」**。
  </Step>
  <Step>
    ### 首次点击：确认 Token 消耗
    首次点击会弹出 Toast：

    > ⚠️ Token 消耗提醒：此操作会将所有视频标题发送给 AI 进行分析，可能消耗大量 Token。再次点击确认执行。

    **5 秒内再次点击** 同一按钮，扩展才会真正执行；超时则需要重新点击一次进入确认态。
  </Step>
  <Step>
    ### AI 分析 + 执行移动
    扩展会：

    1. 拉取默认收藏夹中的全部视频（自动分页）；
    2. 将视频标题与收藏夹名称打包发送给 AI；
    3. 解析 AI 返回的 `[{title, targetFavorite, reason}]` 数组；
    4. 将命中的目标收藏夹不存在的视频 **兜底归入默认收藏夹**；
    5. 按解析结果逐条调用移动接口，每条之间间隔 100ms。
  </Step>
  <Step>
    ### 查看结果
    完成后弹出结果面板，展示每条视频被移动到哪个收藏夹；Toast 同时汇总 **成功 / 失败 / 兜底** 三项计数，并附带每条被兜底的视频详情。
  </Step>
</Steps>

<Callout type="info">
  **执行过程中想取消？** 遮罩层内提供 **「取消」** 按钮，会中断 AI 请求与后续移动操作，但已经移动成功的视频不会被回滚。
</Callout>

## 两种方式如何选

| 维度 | 通过关键字整理 | 🤖 AI 整理 |
|------|----------------|------------|
| Token 消耗 | 无 | 有（与标题数量成正比）|
| 规则维护 | 需要你自己整理关键字 | 不需要，交给 AI 判断 |
| 准确性 | 取决于关键字质量 | 取决于模型能力 |
| 兜底策略 | 未命中视频保留在默认收藏夹 | AI 返回无效目标时自动归入默认收藏夹 |
| 离线可用 | ✅ 完全本地 | ❌ 需要访问模型 API |

## 匹配机制（关键字整理）

- **源收藏夹固定为默认收藏夹**：扩展以你的默认收藏夹作为起点；
- **命中规则**：视频标题命中任一目标收藏夹的关键字即被归入该收藏夹；
- **多重命中**：若标题命中多个收藏夹，按 **关键字管理** 中收藏夹的顺序取第一个；
- **未命中处理**：未命中任何关键字的视频 **保留在默认收藏夹**，不会被误移动。

## 注意事项

<Callout type="warn">
  - **只能从默认收藏夹出发**：扩展不提供选择「源收藏夹」的 UI；如需从其他收藏夹发起整理，请先在 B 站官网把对应收藏夹设为默认；
  - **无预览模式**：扩展不会在正式执行前单独提供预览；建议在大规模整理前 **先把部分视频剪到一个只含几十条的测试收藏夹** 上做小范围试跑；
  - **无进度条**：执行中只有遮罩 + loading 动画，没有分条进度；执行时间取决于视频数量与服务商响应。
</Callout>

<Callout type="info">
  **如何撤销误操作？** 扩展没有内置「一键撤销」。请：

  1. 在 B 站官网打开被误移动的目标收藏夹；
  2. 手动将视频再次移回原收藏夹；
  3. 回扩展修正关键字规则，避免下次再错。
</Callout>
