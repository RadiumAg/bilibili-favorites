## 推荐阅读顺序

- **新用户**：建议先阅读 [收藏夹分析](/docs/guide/favorites-analysis) 了解数据全貌，再通过 [关键词管理](/docs/guide/keyword-management) 与 [批量整理](/docs/guide/batch-organize) 完成首次整理；
- **想要自动化**：配置 [AI 智能分类](/docs/guide/ai-classification) 即可让扩展替你生成关键词；
- **喜欢手感操作**：直接进入 [可视化拖拽](/docs/guide/drag-and-drop) 手动微调；
- **日常使用**：推荐开启 [侧边栏模式](/docs/guide/side-panel)，获得最佳浏览体验。
