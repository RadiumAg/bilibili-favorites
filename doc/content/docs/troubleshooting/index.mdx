## 通用自检

在进入任何具体子页面之前，请先完成以下三项通用自检 —— 它能解决约 60% 的偶发问题：

1. **确认已在 B 站登录**：打开任意 `bilibili.com` 页面，确保账号已登录；
2. **确认扩展是最新版**：在 `chrome://extensions/` 查看版本号，若不是 Chrome Web Store 的最新版本，先更新；
3. **刷新 B 站页面 + 重新唤起扩展**：很多问题在 Cookie 重新建立连接后自动消失。

若上述步骤无法解决，请按类型进入对应子页面继续排查。
